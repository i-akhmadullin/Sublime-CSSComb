Вы успешно поставили плагин CSScomb для Sublime Text 2.


	Плагин сортирует в заданном порядке CSS-свойства в рамках каждого блока правил CSS-кода с помощью алгоритма [CSScomb](http://csscomb.ru).
	Можно выбирать режим работы плагина:
		- онлайн режим (нужна только сеть)
		- оффлайн режим (нужно установить PHP)


Требования к окружению

	(Для работы оффлайн) PHP-интерпретатор. На OS X он установлен из коробки. А для Linux/Windows его нужно скачать с официального сайта [php.net](http://windows.php.net/download/) и добавить путь до папки куда вы установили PHP в переменную окружения $PATH.
	Подробнее о том, как это сделать на Windows читайте - https://github.com/miripiruni/CSScomb/blob/master/src/plugins/csscomb.notepad_plus_plus/README.mkd.


Использование

	ctrl + shift + c,  или в меню Tools → Sort via CSScomb, либо Sort via CSScomb в контекстном меню.

	По умолчанию сортируется весь файл. Так же можно выбрать строки со стилями, которые надо отсортировать.
	Поддерживается множественное выделение. Выделять желательно блок правил целиком, с селектором и скобками.


Переключение режима работы плагина(онлайн/оффлайн)

	Выбрать: Preferences → Package Settings → CSScomb → Settings - Default
	В открывшемся файле поправить в строке:
	"sorter": "heroku"
	значение после двоеточия на "heroku" - для онлайн-режима или "hosted" - для оффлайн-режима.
